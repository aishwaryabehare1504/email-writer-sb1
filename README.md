# AI Email Response Generator

## 🚀 Features
- Generate AI-based email replies from user input
- Supports multiple tones (formal, casual, professional)
- RESTful APIs for generating responses
- Input validation and global exception handling

## 🛠 Tech Stack
- Java
- Spring Boot
- REST APIs
- Gemini API
- WebClient
- Postman

## 📌 API Endpoints
- POST /generate-email → Generate email response

## ▶️ How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Add your Gemini API key in `application.properties`
4. Run the Spring Boot application
